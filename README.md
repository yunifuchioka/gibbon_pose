# File structure
Expects an extra `dataset` folder that the user should manually add to the directory:
```
gibbon_pose/
    dataset/
        train/
            train_0000000.jpg
            ...
            train_0066916.jpg
        val/
            val_0000000.jpg
            ...
            val_0022305.jpg
        train_annotation.json
        val_annotation.json
```
